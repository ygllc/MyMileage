Thanks for considering contributing to MyMileage! Please follow these steps:

1. Fork the repository.
2. Create a feature branch:
   ```bash
   git checkout -b feat/your-feature
   ```
3. Commit changes with clear messages and push to your fork.
4. Open a pull request explaining the change and any testing steps.

### Guidelines
- Keep commits small and focused.
- Add or update unit/UI tests where applicable.
- Update the wiki/README if you change behavior or configuration.

### Code style
- Follow Kotlin conventions.
- Use Android Jetpack libraries where sensible.
